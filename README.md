<h1>Desktop Archiver</h1>

<p>A simple tool that cleans up your desktop from icon clutter and moves those files (based on extensions) to a specified folder</p>

<p>Following file extensions are included:</p>

<ul>
  <li>.jpg</li>
  <li>.jpeg</li>
  <li>.png</li>
  <li>.mp4</li>
  <li>.zip</li>
  <li>.gif</li>
  <li>.pdf</li>
  <li>.txt</li>
  <li>.mp3</li>
  <li>.rar</li>
  <li>.webp</li>
  <li>.webm</li>
</ul>
